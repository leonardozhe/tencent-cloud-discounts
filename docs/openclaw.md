### 详细步骤

#### 1. 购买服务器

1. 访问 [腾讯云优惠入口](https://curl.qcloud.com/Jmr2j2eu)
2. 选择轻量应用服务器
3. 选择配置（推荐 4核4G3M）
4. 选择镜像：OpenClaw / Hermes / DeepSeek-TUI
5. 完成购买

#### 2. 初始化应用

1. 进入轻量应用服务器控制台
2. 找到已购买实例
3. 点击"应用详情"
4. 完成初始化配置

#### 3. 配置模型

```bash
# 配置 TokenHub API（推荐）
export OPENAI_API_KEY=***
export OPENAI_BASE_URL=https://api.lkeap.cloud.tencent.com/v1

# 或配置其他模型
export OPENAI_API_KEY=your-api-key
```

#### 4. 配置消息渠道

支持的消息渠道：
- 微信
- 企业微信
- 钉钉
- 飞书
- Telegram
- Discord
- Web

#### 5. 开始使用

```bash
# 访问 OpenClaw Web UI
http://your-server-ip:3000

# 或通过消息渠道对话
# 直接在微信/钉钉/Telegram 中与 AI 助理对话
```
