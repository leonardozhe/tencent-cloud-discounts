### 配置 OpenClaw

```bash
# 安装 OpenClaw
npm install -g openclaw

# 配置 TokenHub API
export OPENAI_API_KEY=***
export OPENAI_BASE_URL=https://api.lkeap.cloud.tencent.com/v1

# 启动 OpenClaw
openclaw start
```

### 配置 Cursor

```
1. 打开 Cursor 设置
2. 选择 Models
3. 添加自定义 API：
   - Base URL: https://api.lkeap.cloud.tencent.com/v1
   - API Key: your-tokenhub-api-key
4. 选择模型：hy3-preview / deepseek-v4-pro / kimi-k2.6
```
