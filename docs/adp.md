## 核心技术能力

### 1. 工作流生成

```
自然语言描述 → AI 自动拆解 → 完整工作流结构

示例：
"当用户咨询退款时，先查询订单状态，如果已发货则转人工，
如果未发货则自动处理退款"

↓ AI 自动生成

工作流节点：
1. 意图识别：退款咨询
2. 订单查询：调用订单 API
3. 条件判断：是否已发货
4. 分支路由：
   - 已发货 → 转人工客服
   - 未发货 → 自动退款处理
```
