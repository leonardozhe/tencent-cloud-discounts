## 🚀 5 分钟快速体验

### 1. 领取免费额度

选择你感兴趣的产品，点击对应链接领取免费额度：

- **AI 大模型** → [领取 100 万 Tokens](https://curl.qcloud.com/CrDPcagq)
- **轻量服务器** → [领取 1 个月免费](https://curl.qcloud.com/0MIVctv6)
- **语音识别** → [领取免费额度](https://curl.qcloud.com/2OWnJQnv)
- **音视频通话** → [领取免费额度](https://curl.qcloud.com/2GVRH2aZ)
- **游戏服务器** → [查看专属优惠](https://curl.qcloud.com/zX0WAi5D)

### 2. 注册腾讯云账号

1. 访问 [腾讯云官网](https://cloud.tencent.com/)
2. 点击"免费注册"
3. 完成手机号验证
4. 完成实名认证（个人/企业）

### 3. 开通产品

1. 进入控制台
2. 选择对应产品
3. 点击"开通服务"
4. 配置基础参数

### 4. 开始使用

根据产品文档开始使用：
- API 调用 → 获取密钥，调用接口
- 服务器 → 选择镜像，部署应用
- 音视频 → 下载 SDK，集成代码
